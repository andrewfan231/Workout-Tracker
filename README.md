# Andrew's Workout Tracker

A premium-feeling workout tracker for logging training, setting goals, earning XP, redeeming real-life rewards, and checking progression.

Live site:
https://andrewsworkouttracker.netlify.app/

## What It Does

- Create workout templates
- Starter templates for strength, bodyweight, basketball, mobility, conditioning, and general fitness
- Get a smart Home command center with the best next action
- Show a beginner-friendly Start Here path on the Home page
- Load a recommended starter workout from the Profile page
- Log workouts with timers, rest countdowns, effort rating, and notes
- Review and copy completed workout summaries from Past Workouts
- Track simple body check-ins from the Profile page
- Graph bodyweight trends on the Progression page
- Track streaks, XP, goals, shop rewards, and progression
- Add custom rewards to personalize the XP shop
- Show workout cards with focus, estimated time, target volume, and next scheduled day
- Save data in the user's browser for the current version
- Export, import, or clear local data from the Profile page
- Collect beta feedback through Netlify Forms
- Install like an app on supported browsers

## How To Update The Live Site

1. Edit the files.
2. Upload the changed files to GitHub.
3. Click **Commit changes**.
4. Netlify automatically redeploys the site.
